## 4. 字段统一规范

| 规范 | 说明 |
|------|------|
| 主键 | `id BIGINT AUTO_INCREMENT PRIMARY KEY` |
| 创建时间 | `create_time DATETIME DEFAULT CURRENT_TIMESTAMP` |
| 更新时间 | `update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP` |
| 逻辑删除 | `deleted TINYINT DEFAULT 0` (0存在 1删除) |
| 状态字段 | `status TINYINT DEFAULT 0` |
| 备注 | `remark VARCHAR(500)` |

## 5. 状态码约定

| 值 | 通用含义 | 文章 | 批次 | 讲座 |

| 0 | 正常/草稿/禁用 | 草稿 | 生产中 | 草稿 |
| 1 | 待审核/启用 | 待审核 | 已完成 | 报名中 |
| 2 | 已通过/已发布 | 已发布 | 已失效 | 报名截止 |

## 7. 已建立的关键关联

```sql
-- 溯源批次关联农场
trace_batch.farm_id → farm_enterprise.id
trace_batch.land_id → farm_land.id
trace_batch.crop_id → farm_crop.id

-- 专家档案关联用户
knowledge_expert_profile.user_id → sys_user.id

-- 溯源企业档案关联用户
trace_enterprise_profile.user_id → sys_user.id

-- 溯源产品关联农场
trace_product.farm_id → farm_enterprise.id
trace_product.crop_id → farm_crop.id

-- 文章关联作物
knowledge_article.crop_id → farm_crop.id

-- 问题关联作物
knowledge_question.crop_id → farm_crop.id
```
