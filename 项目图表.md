```mermaid
    SysUser --> SysRole : 用户角色
    SysUser --> FarmEnterprise : 管理农场
    FarmEnterprise --> FarmLand : 拥有地块
    FarmLand --> FarmEnvData : 产生环境数据
    FarmDevice --> FarmEnvData : 采集数据
    FarmCrop --> TraceProduct : 关联产品
    TraceProduct --> TraceBatch : 生成批次
    TraceBatch --> TraceRecord : 形成溯源记录
    TraceBatch --> TraceChainBlock : 链上存证
    SysUser --> KnowledgeArticle : 发布文章
    SysUser --> KnowledgeQuestion : 提问
    KnowledgeQuestion --> KnowledgeAnswer : 回答
    KnowledgeQuestion --> KnowledgeYoloResult : 智能识别
    KnowledgeArticle --> KnowledgeDocument : 建立索引
    KnowledgeDocument --> KnowledgeDocumentChunk : 文档切片
```

## 四、技术架构图

```mermaid
flowchart TB
    subgraph Client[用户端]
        Browser[浏览器]
        Admin[管理端用户]
        Farmer[农户/专家/审核员]
    end

    subgraph Frontend[前端层 Vue 3]
        Vue[Vue 3 应用]
        Router[Vue Router 路由权限]
        Pinia[Pinia 状态管理]
        Element[Element Plus 组件]
        ECharts[ECharts 可视化]
        Axios[Axios 请求封装]
    end

    subgraph Backend[后端服务层 Spring Boot]
        Security[Spring Security + JWT]
        Controller[REST Controller]
        Service[业务 Service]
        Mapper[MyBatis-Plus Mapper]
        Exception[统一响应与异常处理]
        OpenAPI[SpringDoc OpenAPI]
    end

    subgraph Business[业务模块]
        System[系统管理模块]
        Farm[农场管理模块]
        Trace[农产品溯源模块]
        Knowledge[农技知识模块]
        Recommend[推荐模块]
        YoloModule[病虫害识别模块]
        RagModule[RAG 检索模块]
        Dashboard[首页统计模块]
    end

    subgraph AI[智能服务层]
        Roboflow[Roboflow YOLOv8 API]
        LocalYolo[本地 YOLO FastAPI 可选]
        Agent[AI Agent 诊断服务]
        RagSearch[文章索引与片段检索]
    end

    subgraph Data[数据存储层]
        MySQL[(MySQL 8.0)]
        FileStore[图片/视频/文件目录]
        ChainData[溯源链式存证数据]
    end

    Browser --> Vue
    Admin --> Browser
    Farmer --> Browser

    Vue --> Router
    Vue --> Pinia
    Vue --> Element
    Vue --> ECharts
    Vue --> Axios

    Axios --> Security
    Security --> Controller
    Controller --> Service
    Service --> Mapper
    Controller --> Exception
    Controller --> OpenAPI

    Service --> System
    Service --> Farm
    Service --> Trace
    Service --> Knowledge
    Service --> Recommend
    Service --> YoloModule
    Service --> RagModule
    Service --> Dashboard

    Mapper --> MySQL
    Trace --> ChainData
    Knowledge --> FileStore
    YoloModule --> FileStore
    YoloModule --> Roboflow
    YoloModule --> LocalYolo
    Knowledge --> Agent
    RagModule --> RagSearch
    RagSearch --> MySQL
```
